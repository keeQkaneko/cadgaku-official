keeQについて

「皆さまが理想のキャリアを実現させるための鍵（key）になる」

株式会社keeQ（キーク）の社名は、未知の扉を開く「鍵（Key）」と、私たちの行動指針である「Quest（探求）」「Question（問い）」「Quality（品質）」という3つの頭文字「Q」を組み合わせた造語です。この社名には、「常に探求心と問題意識を持ち、品質にこだわる」という理念が込められています。

今回、私たちが完全無料の就職直結型CADスクール「キャドガク」を立ち上げた背景には、「挑戦の格差をなくす」という思いから実現しました。 「新しいスキルを身につけてキャリアを変えたい」という強い熱意を持ちながらも、高額な受講料といった経済的・環境的な理由による、教育の機会損失、将来への不安はあるものの、何から始めて良いかわからず、一歩を踏み出せない20代の若者が数多く存在します。一方で、日本のものづくりやインフラを支える産業界では、深刻な技術者不足が大きな課題となっています。

私たちは、この「挑戦したい若者」と「若手の技術者人材を求める企業」の間に生じるギャップを埋めたいという想いから、キャドガクの運営をスタートしました。 学びを諦めない環境をつくり、誰もが正当に評価され、望むキャリアに挑戦できるよう、株式会社keeQは、キャドガクを通じて、人と企業の未来を切り拓く最高のパートナーであり続けます。

Value 01

挑戦の格差をなくす

経済的・環境的な理由による教育の機会損失をなくし、誰もが平等に学べる環境を提供します。

Value 02

本質的なスキルの育成

形だけの資格取得ではなく、企業が今まさに現場で求めている「生きたスキル」を徹底教育します。

Value 03

一気通貫の伴走支援

教育から就職まで、一人の生徒にプロ2名体制で寄り添い、伴走します。
